# 一

## 抓型流程

熟悉后，可以删减步骤；**用直线，少转折**。

1. 一阶段（一级型）

   - 外轮廓（黑色）
   - 中线（头、身体、腿；腿如果分开，选一边）
   - 比对比较

2. 二阶段（二级型）

   - 切负型（红色）
   - 切大结构（蓝色）
   - 比对比较

3. 三阶段（查漏补缺，绿色）

4. 四阶段（勾线）

5. 五阶段（细节）

   - 主要褶皱（黑色；==注意疏密、不要等距、不要平行==）
   - 色块（可以少量漏色，有透气感）

   - 闭色（光进不去的地方，少量）

## 头发

### 问题

- 结构
  - 头型 + 头发厚度
  - 不跟透视转面
- 构成
  - 铺陈线条（少）
  - 等分（不要等分、要大中小）
  - 黑白区分（旋转）
  - 细节不够
  - 碎
  - 巧合对接

### 流程

- 外轮廓（整）
- 短发（分位置发组、按组分发片穿插、按透视+细节）
- 起始位（发际线 》眉心 = 眉心 》鼻头）
- 卷发（翻转）
- 构成（疏密套娃，破）
