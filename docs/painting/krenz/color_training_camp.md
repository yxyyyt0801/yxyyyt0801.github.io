# 一

## 基本概念

- “固有色”指物体在均匀白光(如自然光)下呈现 的物体固有色彩。如香蕉的固有色是黄色，苹果的 固有色是红色。

- 二分

  - 暗、亮、阴影 》立体感
  - 暗示结构
  - 一束**直射光**照射到物体后，形成**高度概括**的“亮”与“暗”的效果
  - 固有色 》几何二分 》粗二分 》精二分 》完成（暗中暗、亮中暗、环境光、高光）
  - 观察
    - 优先看**投影**，其次看亮暗占比
    - 判断最强主光源方向即可

- HSB

  - 色相（颜色）

    - H（Hue）
    - 0~360环收尾相连
    - 表示颜色的基本类型，如红、绿、蓝（PC绘画三原色，光学三原色）。取值范围通常是 0-360 度，对应色轮上的角度
      - 传统三原色：红、黄、蓝，传统绘画里画材为颜料，这三个颜色无法通过混合其他颜色得到

  - 饱和度（鲜艳、纯度）

    - S（Saturation）

    - 0~100

    - 表示颜色的纯度。0% 为灰色，100% 为最鲜艳的颜色。

  - 明度（明暗）

    - B / V（Brightness/Value）
    - 0~100
    - 表示颜色的相对明亮程度。0% 为纯黑，100% 为最亮状态。

- 配色优化方案

  - 黑白关系拉开(B)
  - 确定主色（H，领居配色技巧）
    - 色调统一（柔光）
      - 新建图层（剪贴模板，模式选择柔光）
      - 合并黑白图层
      - 色相饱和度（ctrl+u）提高饱和度（如果感觉发灰）
    - 肤色固定
    - 领居色（15度内）
  - 添加高饱和（S）或对比色（H）抢眼
    - 点缀色
    - 深入细化
  - 花纹和渐变有奇效！



## 上色流程







# Ps技巧

- 颜色窗口 | HSB滑块 | 变长、变窄（可以用于**抓色练习**）
  - 观察目标色,在颜色面板中优先调整H(色相) 接近参考色,S(饱和度)和 B(明度) 先定位50% 
  - 对比观察初步取色与目标色的差异,再微调各项数值反复取色对比,直到自 己所取颜色接近目标色
  - 每项数值差距<10 则视为通过
- 正片叠 底：混合加深
  - 如果【正常模式】中的颜色为纯白色,启用【正 片叠底】后,白色部分不起效,显示为透明。
- 剪贴蒙版
- 锁定图层
- 锁定透明像素（阿尔法锁定）
- 笔刷（Krenz-笔刷.abr）
  - 设置：小缩览图，每排10个
  - 推荐笔刷
    - 5（第一个）“尖角5像素”笔刷（硬边圆），粗细随笔轻重变化，不透明度不变
    - 57（5下面的）19号，不透明度随笔轻重变化，粗细不变
    - 246（第二行第四个）柔角笔刷（喷枪），不透明度随笔轻重变化，边缘偏模糊
  - 笔刷参数
    - 笔刷粗细，自行调整
    - 混合模式，正常
    - 不透明度：100%
    - 大小压力：不点选

- 查看明度，新建图层（剪贴蒙版），模式选择饱和度，填充纯白、纯黑
