# 快捷键

| 快捷键  | 说明                            | 扩展                                                         |
| ------- | ------------------------------- | ------------------------------------------------------------ |
| b       | 画笔                            | 创建文档<br />shift+lmb 画笔大小<br />[ & ] 画笔大小<br />ctrl+lmb 吸色<br />i & o 不透明度<br />k & o 颜色深浅<br />e 橡皮 |
| f       | 填充选区（前景色）              |                                                              |
| t       | 移动工具（位图）                |                                                              |
| ctrl+t  | 变形工具                        |                                                              |
| ctrl+g  | 图层打组                        |                                                              |
| ctrl+j  | 复制图层                        |                                                              |
| ctrl+e  | 合并图层                        |                                                              |
| r+lmb   | 选取图层                        |                                                              |
| delete  | 删除图层内容                    | shift+delete 删除图层                                        |
| d       | 默认前景色 “黑色”               |                                                              |
| h       | 历史颜色                        |                                                              |
| u       | 常用颜色（ “显示图像采样色块”） |                                                              |
| shift+j | 椭圆工具                        | 按住 shift 正圆<br />按住 alt 移动                           |
