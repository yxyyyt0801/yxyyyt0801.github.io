# 工程和资产

## 工程

### 项目结构

项目路径，如：D:\data\opentoonz\anim

- 项目总目录
- 有多个项目子目录，如：ep01、ep02



项目

- 一个项目就是一个镜头，只放一个场景，方便备份、打包、工程交接
- 在项目子目录下创建项目，如在ep01下创建项目
- 项目名称，如：a_ep01_001、a_ep01_002
- 文件目录
  - drawings（不同图层一个编号文件；每个图层都有，tlv光栅、tpl色板、pli矢量）
  - extras（外部素材）
  - inputs
  - outputs
  - palettes
  - scenes
    - 场景名称.tnz（场景文件，本质是一个时间轴）
  - scripts
  - otprj.xml（项目文件）



场景

- 场景名称，如：1
- 分辨率：1920 * 1080（2338 * 1654）
- dpi：53.34（对抗矢量显示光栅的bug）
- 帧速率：24



### 切换项目

通过加载不同场景来切换项目；文件 | 加载场景

- 选择 场景名称.tnz
- 更改项目



### 备份项目

- 文件 | 导出 | 导出当前场景
- 直接备份 a_ep01_001 目录



## 资产

### 何为资产

被tnz抓取的文件为资产，可以被同一个图层共享。如：

- drawings 目录的 tlv光栅、tpl色板、pli矢量
- extras 外部素材



### 查看资产

- 层级窗口



### 导入资产

- 外部文件导入

  - 管理窗口 拖文件（图片、视频） 到时间轴

  - **导入**（复制到extras ）
    - 如果是视频资产，层级窗口就会看到序列帧
      - 放大，用于**转描**

- 层级窗口导入其他图层帧，然后导出

  - 全选帧 | 替换层级 | 替换源 | 选择层级（注意不能是多个不同图层资产）



### 导入序列帧

- 重命名 a.4位数字.png
- 第一张图拖到时间轴
- 导入



### 资产合并

- A层，右键 | 拷贝到；右键 | 插入到上面
- A层，全选层级，右键 | 替换层级 | 替换层级 | 导入（添加后缀来重命名新文件）
- 多选两个图层 | 摄影表，合并tlv光栅层级
- 删除不完整图层列
- 层级，移除全部未使用层级（A层，右键 | 替换源 确认）



# 术语

- layout（构图）：根据粗略分镜决定背景构图及角色的动作和配置，然后画下来进行更加细致的设计图
- 一原：第一原画简称。即草原画，一般只有演技动作，人物的服装、表情、细节等等都没有细化
- 分色线：红色线的范围表示高光，蓝色表示阴影，绿色表示二重阴影，为上色做区分
- 中割：把原画间的动作画全，是整个动画的主要部分，也称为中间画
- 轨目：在中割时的标识，上面标注有关键帧、帧数目和动画中对象运动的速度，动画人员依据标识画出中间画
- 律表：又叫摄影表，将动作的时间节奏以格数展示的表格，通常一秒24格
- OT文件格式
  - 光栅格式 raster formats：tif，tga，png
  - Toonz光栅格式 toonz raster formats：tlv
