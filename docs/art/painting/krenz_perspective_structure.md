# 透视

摄影机

- 镜头范围（视野）

    - 一点透视（正面）

    - 二点透视（侧面）

  - 摄像机垂直旋转
    - 三点透视

- **几点透视是相对于被拍物体**

## 建模

三视图

- 主视图，由前向后看
- 俯视图，由上向下看
- 左视图，由左向右看



模型难度：方体、曲面物体、透视中的对称
