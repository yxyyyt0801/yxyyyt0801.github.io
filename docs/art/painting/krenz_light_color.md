# 光影

## 固有色 * 环境光色 * 直射光色

### 原理

1. 软光源（漫反射，摄影棚，面光源）-> **固有色**（白光，物体本身颜色）
2. **环境光色**：环境调暗 -> 压暗环境和物体
   1. 亮度
   2. 色系
   3. 饱和度
3. **直射光色**：开灯 -> 物体，亮暗面
   1. 亮度
   2. 色系
   3. 饱和度



### 图层

1. 背景（普通图层）
2. 线稿+配色（普通图层）
3. 压暗@灰色（剪辑图层 + 正片叠底）
   1. 正片叠底区域是暗色系
   2. 色彩平衡（ctrl+b），色彩微调（偏色）
   3. 色相/饱和度（ctrl+u），明暗，饱和度，色相微调（想换个颜色，或者去灰）
4. 开灯@灰色（剪辑图层 + 颜色减淡）
   1. 颜色减淡区域是光色系；明度越高，光照越强
   2. 色彩平衡（ctrl+b），色彩微调
   3. 色相/饱和度（ctrl+u），明暗，饱和度，色相微调
