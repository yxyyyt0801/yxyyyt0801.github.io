# 简介

注册会计师（CPA）的全称是 **Certified Public Accountant**。

其中文全称为 **注册会计师**，这是依据《中华人民共和国注册会计师法》设定的国家级执业资格。

**法定资格**：唯一官方认可的注册会计师执业资质，受《注册会计师法》保护。

**签字权**：拥有在审计报告上签字的法定权利，是审计、证券、金融等领域的核心门槛。

**认可度**：在会计师事务所、大型企业、金融机构、政府监管单位中被高度认可，是财务高管岗位的优先条件。



# 要求

专业阶段（6科）

须**同时满足**：

- 具有完全民事行为能力（年满18周岁，或16周岁以上以自己劳动收入为主要生活来源）
- **学历/职称二选一**：高等专科及以上毕业学历（全日制/自考/成考/函授等均认可，学信网可查），**或**会计/审计/经济/统计等相关专业**中级及以上技术职称**（如中级会计师）
- **不限制专业、不要求工作年限**，本科大四/专科大三可按应届毕业生身份预报名



综合阶段（1科）

须**同时满足**：

- 具有完全民事行为能力
- 已取得注册会计师专业阶段考试**全科合格证**（专业阶段单科成绩5年内有效）



不得报名情形

- 被吊销注册会计师证书自处罚日起不满5年
- 因考试违规尚在禁考期内
- 已取得全科合格证或已被依法认定具有注册会计师资格者

# 科目

## 考试科目

**专业阶段（6科）**：会计、审计、财务成本管理、税法、经济法、公司战略与风险管理

**综合阶段（1科）**：职业能力综合测试（试卷一+试卷二），需专业阶段6科全过后才能报考



## 成绩有效期

**专业阶段（6科）— 单科成绩 5 年内有效**

- 从你**第一科通过的那一年起算**，须在**连续的5个考试年度内**通过其余5科。
- 举例：2026年过了《会计》，最晚要在 **2030年** 把剩下5科全部考过；如果2030年仍有未过科目，《会计》成绩失效需重考。
- 之后每新通过一科，都会重新带动"5年滚动窗口"往前推。

**综合阶段（1科）— 无期限限制**

- 专业阶段6科全过取得**专业阶段合格证**后，**综合阶段不限时间**，任何时候都能报考，成绩长期有效。
- 考过综合阶段后可申请成为**非执业会员**。

CPA**全科合格证书本身不会过期**，一旦考过专业阶段6科+综合阶段拿到《注册会计师全国统一考试全科合格证书》，这个资格终身有效，不用年审也不会被收回。



考完CPA后你通常要申请加入中注协成为**会员**，分两种情况：

非执业会员（大多数考CPA的人走的路）

- ✅ **证书不作废**，但需： **每年参加继续教育**（一般≥40学时/年，分面授+网络） **按时缴纳会费**（各地标准不同，通常每年几十到百余元） 在**省注协规定时间内完成年度检查/确认**
- ❌ 若长期不交费、不参加继续教育且不办理保留手续，会被**注销非执业会员资格**——但全科合格证仍在，补齐条件可申请恢复。

执业会员（在事务所从事审计满2年申请后获得）

- 需**在事务所连续执业**、完成继续教育、缴会费、通过注协年检
- 离开事务所满**2年**自动转为非执业会员，不会直接"作废"



## 满分与合格标准

专业阶段6科**每科独立计分**，满分100分，60分即通过。

综合阶段虽分为两卷，但**合并计分**，总分100分，60分合格。

所有科目均实行**百分制**，60分为统一的合格分数线。

**全部机考**：考试采用闭卷、计算机化考试方式，考生在计算机终端获取试题、作答并提交答题结果。

# 计划

## 第1年：会计 + 审计 + 税法

### 阶段一：基础搭建（12月‑次年4月）

| 科目     | 学习顺序                        | 每日时间 | 重点章节（参考）                                   | 目标                                       |
| -------- | ------------------------------- | -------- | -------------------------------------------------- | ------------------------------------------ |
| **会计** | 1. 先学                         | 2‑2.5h   | 第6章 长期股权投资、第17章 收入、第27章 合并报表等 | 掌握60%基础概念，理解分录逻辑              |
| **审计** | 2. 接续（会计学完3章后开始）    | 1.5‑2h   | 第1‑8章（审计基本原理）、第7章 风险评估等          | 建立审计流程框架，理解与会计的关联         |
| **税法** | 3. 同步（会计所得税章节后开始） | 1‑1.5h   | 第2章 增值税、第4章 企业所得税                     | 熟悉主要税种计算，注意与会计利润的调整关系 |

**材料**：官方教材 + 《应试指南》/《轻1》基础班课程。



### 阶段二：强化突破（5月‑7月）

- **会计**：主攻重难点章节（如合并报表、金融工具），刷《必刷550题》。
- **审计**：结合案例理解循环审计、特殊事项审计（第9‑17章）。
- **税法**：集中练习增值税、企业所得税综合题，整理税收优惠。
- **时间配比**：会计:审计:税法 ≈ 4:3:3。



### 阶段三：冲刺模考（8月‑考前）

- **真题实战**：每周完成1‑2套历年真题，严格计时。
- **机考适应**：使用中注协机考模拟系统练习。
- **错题复盘**：重点回顾高频错题知识点。
- **时间配比**：税法冲刺为主（占50%），审计30%，会计20%。



## 第2年：财管 + 经济法 + 战略

### 阶段一：基础期（1月‑4月）

| 科目       | 学习顺序                       | 每日时间 | 重点章节                                 | 目标                           |
| ---------- | ------------------------------ | -------- | ---------------------------------------- | ------------------------------ |
| **财管**   | 1. 先学                        | 2h       | 第3章 价值评估基础、第10章 本量利分析等  | 理解公式推导，建立计算模型思维 |
| **经济法** | 2. 同步（财管学完1‑2章后开始） | 1h       | 第4章 合同法、第6章 公司法、第7章 证券法 | 熟悉法条框架，结合案例理解     |
| **战略**   | 3. 后期加入（5月后）           | 0.5‑1h   | 第2章 战略分析、第3章 战略选择           | 掌握分析工具（如PEST、SWOT）   |

**材料**：官方教材 + 《应试指南》/《轻1》 + 经济法法条汇编。



### 阶段二：强化期（5月‑7月）

- **财管**：大量刷题提升计算速度，重点练习资本预算、成本分析。
- **经济法**：归纳对比公司法、证券法等重点法条，做案例题。
- **战略**：学习将理论应用于案例分析，背诵核心模型。
- **时间配比**：财管:经济法:战略 ≈ 5:3:2。



### 阶段三：冲刺期（8月‑考前）

- **财管**：机考练习（计算题输入速度是关键）。
- **经济法**：集中背诵重点法条，做主观题。
- **战略**：强化案例题训练，背诵高频考点。
- **模考**：每周全真模拟，适应三科连考节奏。



## 第3年：职业能力综合测试（试卷一+试卷二）

**考试内容**：试卷一（鉴证业务，侧重会计、审计、税法）+ 试卷二（管理咨询，侧重财管、战略、经济法）。

**备考核心**：**整合能力**，而非新知识。

**建议安排**：

- **3‑6月**：回顾专业阶段重点，使用《综合阶段辅导教材》建立跨科大题思维。
- **7‑8月**：做历年综合真题，练习**案例分析与职业判断**。
- **考前**：参加模考班，适应一天两卷的强度。

# 职业规划

**科技公司财务分析/经营分析(FP&A)或 BI财务**

用SQL/Python/Excel做收入成本分析、预算建模，CPA证明你懂准则→起薪和发展都比普通会计好



**ERP财务顾问 / 财务系统实施（SAP/用友/金蝶）**

IT背景+CPA→懂业务又懂系统，甲方乙方都抢，收入可观



**企业内审 / IT审计（SOX合规）**

大中型企业或事务所IT审计组，看重IT经历+CPA，不要求年轻



**传统中小企业主办会计（保底选项）**

若想先有一份稳定财务工作过渡，也完全可行
